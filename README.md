Mobile app for tracking your pets' health and wellness.

##  How to Run
```bash
# Install dependencies
npm install
npm install lucide-react-native
npm install react-native-svg

# Start the project
npx expo start
```

Scan the QR code with **Expo Go** app or press `a` (Android) / `i` (iOS).

##  Requirements

- Node.js
- Expo Go (on your phone)
